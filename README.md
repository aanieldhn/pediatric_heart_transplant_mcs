This is the repository containing all statistical code written for the paper "Loss of Acuity-Based Priority in U.S. Pediatric Heart Transplantation." All analyses, figures, and tables that are in the manuscript can be generated with the files that are included in this repository. One must have the 2026Q2 version of the Organ Procurement and Transplantation Network (OPTN) Standard Transplant Analysis and Research (STAR) files, which can be requested by the OPTN. One will also require access to "ptr_hrhl.sas7bdat," a supplemental file from the OPTN that contains all donor heart match runs. To obtain these files requires a priori IRB approval along with a data use agreement with the OPTN. Please run the RMD "pediatric_heart_tx_mcs.Rmd" first. This produces the cohort that is used for the paper and generates Figures 1-3 and Table 1. Next, run "match_run_analysis.Rmd" to run our match run analysis, in which we define a 4-status hypothetical counterfactual system that prioritizes patients supported by ECMO or mechanical ventilation. This file generates Figure 4. The code for generating eTables 1-4 can be found in "Supplemental Tables.R."
